# Система учёта клиентов для малого бизнеса

**Специальность 09.02.07 «Информационные системы и программирование»**

## Функциональность

### Роли пользователей
- **👑 Администратор** — полный доступ (CRUD клиентов, визитов, услуг)
- **✂️ Мастер** — работа с клиентской базой и визитами
- **📊 Владелец** — финансовые отчёты и аналитика

### Сущности (по КР)
- Клиент (ФИО, телефон, email, скидка)
- Услуга (название, цена, длительность)
- Визит (дата, список услуг, сумма, статус оплаты)
- Состав визита (связь многие ко многим)

### Отчёты
- По частоте посещений
- Финансовая аналитика для владельца

## Запуск
Просто откройте `index.html` в браузере.  
Данные хранятся локально в IndexedDB.

## Технологии
- HTML5/CSS3
- JavaScript (ES6+)
- IndexedDB (локальное хранилище)
